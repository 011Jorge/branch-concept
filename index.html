Entendendo o conceito de Branch

Branches são iguais ao multiverso da marvel. Aonde uma branch principal se chama main ou master

Mas como funciona uma branch?

uma branch é aonde você pode criar um novo multiverso a partir da sua master, sem correr o risco de explodir seu multiverso de códigos.

E se caso você quiser juntar as branch do multiverso com a branch original é só juntar.
